# Tetris
## 概要
C++/CLIとWindowsフォームで作ったテトリスです。

## 操作方法
- ←キー・・・ブロックを左に移動
- →キー・・・ブロックを右に移動
- Aキー・・・ブロックを左回転
- Dキー・・・ブロックを右回転
- スペース・・・落下速度を上げる

Debugディレクトリ内のTetris.exeから実行できます。

## 開発環境
- Visual Studio Community 2015
<br><br>


## About
Tetris game with C++/CLI using Windows Forms.

## How to play
- Left arrow ----- Move block to left.
- Right arrow ----- Move block to right.
- A ----- Rotate block to left.
- D ----- Rotate block to right.
- Space ----- Falling speed up.

Execute "Tetris.exe" in the "Debug" directory to play.

## Development Environment
- Visual Studio Community 2015
